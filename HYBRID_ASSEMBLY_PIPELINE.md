# Hybrid Genome Assembly Pipeline (Streptococcus thermophilus)

A high-continuity microbial genomics workflow that leverages the complementary strengths of Illumina (high-accuracy short reads) and Oxford Nanopore (long structural reads) to construct a complete, unfragmented *Streptococcus thermophilus* genome.

## Project Objective
To evaluate, compare, and validate genomic reconstructions of *S. thermophilus*. The pipeline runs a parallel benchmarking analysis: a short-read-only de novo assembly vs. a hybrid long-and-short read co-assembly. Quality and contiguity metrics are structurally evaluated using QUAST.

## Step-by-Step Workflow Execution

### Phase 1: Illumina Short-Read Baseline Assembly
Generate a baseline assembly using only the highly accurate, short insert-size fragments.

```bash
# Execute short-read de novo assembly
spades.py -1 Short_1.fastq.gz -2 Short_2.fastq.gz -o short_assembly

# Evaluate baseline assembly contiguity
quast.py short_assembly/scaffolds.fasta -o quast_out
```

### Phase 2: Hybrid Long + Short Read Assembly
Execute the hybrid assembly. SPAdes utilizes the high-accuracy short reads to construct a de Bruijn graph and fix sequence-level errors, while using the Nanopore long reads to resolve ambiguous repetitive loops and span large genomic gaps.

```bash
# Execute hybrid co-assembly
spades.py -1 Short_1.fastq.gz -2 Short_2.fastq.gz --nanopore ont.fastq -o spades_out

# Evaluate hybrid assembly contiguity
quast.py spades_out/scaffolds.fasta -o quast_hybrid_out
```

## Bioinformatics Insights & Interpretations

### Key Metric Breakdowns:
*   **Contiguity ($N50$ and $L50$):** The Hybrid approach dramatically improved the $N50$ value from **64,398 bp to 196,927 bp**, a nearly 3-fold increase. Simultaneously, the $L50$ drops from 10 to 4, proving that half of the total genome size is safely captured within just 4 ultra-long scaffolds.
*   **Fragmentation Reduction:** Short-read constraints left the genome broken across **55 distinct contigs**. By incorporating Oxford Nanopore reads, the assembly is consolidated down to **14 contigs**, drastically simplifying downstream gene annotation.

### Final Conclusion
The **Hybrid Assembly provides the superior biological result.** 

Short-read graph traversals break down when hitting identical genomic repeats (e.g., rRNA operons or insertion elements) because short pieces cannot uniquely span them. The long Nanopore reads successfully bridge these repetitive regions. While the baseline Illumina reads correct the higher per-base indel error rates typical of older Nanopore runs, the hybrid combination delivers an accurate, nearly complete, and structurally robust blueprint of the *Streptococcus thermophilus* genome.
